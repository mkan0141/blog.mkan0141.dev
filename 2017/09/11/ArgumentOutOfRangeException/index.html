Bundle ID

Bundle IDは[company nameの逆]にしなければいけないみたいです。

例
company name -> mkan.home.company
と設定していれば
Bundle ID -> company.home.mkan
と設定しなければならないみたいです。

bundle id の変更方法は簡単です。

  1. build settings -> player settingsを押す。
  2. Others settingsのBundle Identifier の所の文字を変える
